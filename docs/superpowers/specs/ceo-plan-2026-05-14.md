## Vision

### 10x Check
不是又一个攻略站，而是"PoE2中文玩家的默认工具"。DPS计算器一键分享链接（NGA/贴吧传播），中英数据自动对照匹配英文Build名，推荐系统从静态文章变成交互式探索。

### Platonic Ideal
新玩家打开网站：选职业→看到前中后期推荐路线（带DPS模拟数值）→点进计算器→改装备技能实时看到伤害变化→找到喜欢Build→一键分享→朋友打开链接就能查看和修改→搜"电弧"得到技能详情+关联Build+推荐装备。

## Accepted Scope (added to this plan)
- Build share links (short URL + serialized config)
- Cross-reference linking (skill → builds → equipment)
- Personalized homepage (remember user class/build preferences)
- Build comparison tool (side-by-side DPS/defense compare)

## Deferred to TODOS.md

- Advanced: equipment-to-build recommendation engine
- Advanced: passive tree visual simulator

## Execution Strategy (from office-hours)

## What's NOT in scope (explicitly deferred)
